# go_datastructures

Data structures in Go. Getting my feet wet with the language before building bigger things (see `capradb`).

## What's here

- `ds/arraylist` - dynamic array
- `ds/stack` - stack
- `ds/queue` - queue
- `ds/hashtable` - open addressing hash table
- `ds/lru` - LRU cache
- `ds/btree` - B-tree
